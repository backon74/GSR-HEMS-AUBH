# SmartCool HEMS
**AUBH Team · Energy Track · Global Students Research Hackathon 2026**

A predictive A/C control system that pre-cools homes before electricity demand peaks, adapts to humidity via dew point analysis, and estimates condensate water recovery using anonymised residential data.

## Results
- **30–40%** annual energy savings · **<1 month** payback · **~SAR 900M/year** at Eastern Province scale
- Peak demand reduced by ~28% · Resident comfort maintained 100%

## Run the Dashboard

